🎥🎵 Movie & Music Recommendation System

Tech Stack: Python, Flask, Scikit-learn, Pandas, Spotify API, TMDB API, HTML/CSS

A smart recommendation system that provides users with personalized movie and music suggestions using both content-based and collaborative filtering techniques.

🔍 Features

  • 🎵 Spotify Integration: Recommends music using audio features and user preferences.
  • 🎥 Movie Recommendations: Uses metadata and collaborative filtering to suggest relevant titles.
  • 🧠 Machine Learning: Cosine similarity, matrix factorization (SVD), and KNN-based algorithms.
  • 🌐 Real-time API Integration: Uses TMDB and Spotify APIs to fetch latest data.
  • 🖥️ Deployed with Flask: Responsive UI with clean layout and input interface.
